# 006 Text Embedding Retrieval OFDM Rayleigh

Digital text embedding retrieval over an OFDM link with a Rayleigh block fading propagation model.

Current PHY assumption:
- SISO only (`num_tx=1`, `num_streams_per_tx=1`, single receive antenna)

Run:

```bash
./experiments/006_text_embedding_retrieval_ofdm_rayleigh/run.sh
```

Config:

```text
experiments/006_text_embedding_retrieval_ofdm_rayleigh/config.yaml
```
